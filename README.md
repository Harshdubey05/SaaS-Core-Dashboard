# SaaS Core Dashboard

A premium, high-performance SaaS Dashboard UI built with a focus on modern aesthetics, glassmorphism, and smooth interactions. This project features a complete "SaaS Core" engineering management experience, consolidated into a single, highly-optimized HTML file.

## 🚀 Key Sections

- **Hero Header**: "Your Business, at a Glance." — Instantly see how your growth is scaling with real-time insights.
- **Metrics Dashboard**: 
  - **Revenue**: $124,500 (+12.5%)
  - **Active Users**: 42,891 (+8.2%)
  - **Churn Rate**: 2.4% (-0.5%)
  - **LTV**: $1,850 (+4.1%)
- **Revenue Growth**: Sophisticated interactive area chart visualizing growth over time (7D, 30D, 1Y views).
- **Top Projects**:
  - **Acme Rebrand**: In Progress ($45k)
  - **Core API V2**: Review ($28k)
  - **iOS App Sync**: Planning ($12k)
- **Recent Activity**: Real-time event timeline including subscription alerts, server status, and backup logs.

## 🛠️ Technology Stack

- **Core**: HTML5, Vanilla JavaScript
- **Styling**: [Tailwind CSS](https://tailwindcss.com/) (JIT CDN)
- **Typography**: [Inter](https://fonts.google.com/specimen/Inter) via Google Fonts
- **Icons**: [Material Symbols Outlined](https://fonts.google.com/icons)

## 🎨 Design System

- **Branding**: SaaS Core (Engineering Team)
- **Primary Palette**: 
  - Background: `#0B0B0F`
  - Primary: `#3B82F6` (Blue)
  - Secondary: `#A855F7` (Purple)
  - Tertiary: `#F97316` (Orange)
- **Aesthetics**: Glassmorphism (backdrop-blur), subtle micro-animations, and high-contrast dark mode.
